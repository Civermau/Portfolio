About This Webpage

The Beginning

I wanted to take a moment and write a little about this webpage. The creation of this site has taken months of work, I always was a desktop developer, only using C based languages, later I learned a little java and other languages, but it was all for a general apps or backed. Since I was a kid I was interested in content creation, and now I think that I can share a little on what I know, I started working on all of that, creating social media accounts, writing scripts and getting ready to it.

But for me, to be more close to my image of a real developer and a content creator, I though I needed a website of myself, a place where I can share my projects and designs, or leave my personal presentation.

The Learning Journey

And with that, I started learning web in late 2024, started from scratch, learning html, css and javascript, I created a first version of this site, I learned to use github pages, I bought a domain, I learned to realte that domain to my github pages...

little by little I slowly learned more about web development, I asked in reddit, I asked in stack overflow, I watched tons of youtube videos, I read docs, asked chatgpt for ideas for projects for practicing...

It's been a real adventure, and it has been extremelt fun, other times stressful as hell, but in the end, I liked it. I enjoyed watching how with all of my work, a good looking website can be created.

Growing Experience

A friends of mine started a studio for events, and there I had the opotunity to learn even more since I was one of the people that develop the website for the studio.

And after all of that, I'm still here, creating this website, a new portfolio. The creation of this site was a real challenge, my biggest and most good looking page yet. making this portfolio took my 2 weeks of work, all day for two weeks, it had a lot of sketches, a lot of versions, a lot of work. But eventually I arrived to this version that I'm proud of, something weird, since I rarely feel proud of my work.

I has some indexes that where discarded in the way to the final version, maybe in the future I'll reevaluate them, but for now, I'm happy with the result.

A Friendly Request

But if you just want to make a wget and change my name with yours (Yeah, I am still looking at you, randomike), please, avoid doing it. But at the end, I can't do much if you do it, is not like I'll demand you or something, I'll just be sad. You really want to make me sad?

I'm not a professional web developer, man I am not even a professional normal developer, I'm a self taught one, and I'm still learning, I'm still improving, I'm still making mistakes... But I am happy to learn and improve, day by day, I'm getting better. I'm happy to be here.

If you somehow took the time to read all of this, thank you.
