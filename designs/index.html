Resources

Even if they are not the best, I still want to share what I use and what I have learned to maybe help you, if you want to create your very own designs.

Programs

I use Paint.net for creating everything with BoltBait's Plugin Pack. Sometimes I use krita since it has more tools, but I got used to Paint.net, I have been using it for a long time.

Hardware

I use a small wacom one drawing tablet, it was the cheapest and the only one I could afford, but it is pretty good for what I need. I use it with drawing programs, but even in davinci resolve.

Love, I guess

If I made this, was because I found actual interest and passion for it, works from people I admire inspired me to start my own try, I aim to become better at it.
